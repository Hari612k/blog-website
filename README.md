- CSS
- JavaScript

---

## ✨ Features

- 3 complete blog posts with title, image, date, and content
- Navigation menu to jump to posts
- Comment section (JavaScript-based)
- Search bar to filter blog posts live
- Share button simulation
- Tags for each post
- Fully responsive layout for mobile and desktop

---

## ▶️ How to Run

1. Open `index.html` in any web browser
2. Use the search bar, leave comments, and try the share buttons
